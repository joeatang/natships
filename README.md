# NATSHIPS · Vessels of the Substrate

A non-arbitrary NFT collection for the [nat.fun](https://nat.fun) ecosystem. Every vessel and every sky in this collection is mathematically forced by one Bitcoin block — nothing arbitrary, nothing hand-decided.

**Tagline:** *Ships forged from the chain. Skies forged from the chain. One per block. Built to return.*

---

## What this is

The project fuses three theses into one collection:

| Thesis | Source |
|---|---|
| **Non-arbitrary matter** — assets emerge from on-chain data | Digital Matter Theory (DMT) |
| **Immutability** — block data is permanent, conserved, never destroyed | Bitcoin |
| **Reusability** — vessels are not consumed by their journeys | SpaceX |

All three say the same thing in different vocabularies: **matter / data / mass is conserved. Nothing wasted.**

Every NATSHIP card is a cinematic still — a vessel rendered against a cosmos that's also derived from the same block. The chemistry, the geometry, the celestial structure of the sky — all emerge from the block's hash. The vessel's fleet, class, livery, engines, flight log — also from the block. **A vessel forged from the matter it travels through.**
